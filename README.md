## dd2476 project

clone the repository and run
npm @v5
node @v10

```bash
    npm install
```

and then run

```bash
    npm run dev
```

or

```bash
npm run client
npm run server
```

in two terminals to watch separately client and server changes.

webpack devServer @port 8888
express serves @port 3000

### crawl the data

```bash
cd crawler
npm install
node index.js
```

read the comments in `index.js` file
